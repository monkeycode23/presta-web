# Sistema de Gestión de Préstamos

## Descripción
El **Sistema de Gestión de Préstamos** permite a los usuarios administrar de manera eficiente la información relacionada con clientes, préstamos y pagos, así como generar reportes para el control y seguimiento de las operaciones financieras. El objetivo del sistema es centralizar la gestión de préstamos, facilitar el registro de pagos y ofrecer información clara para la toma de decisiones.

## Funciones principales

### Gestión de Clientes
- Registrar nuevos clientes.
- Modificar información de clientes existentes.
- Eliminar clientes cuando sea necesario.
- Consultar la información de los clientes registrados.

### Gestión de Préstamos
- Registrar nuevos préstamos asociados a clientes.
- Modificar los detalles de los préstamos existentes.
- Cancelar préstamos según corresponda.
- Consultar la información de los préstamos activos o finalizados.

### Gestión de Pagos
- Registrar pagos realizados por los clientes.
- Consultar el historial de pagos de cada préstamo.

### Generación de Reportes
- Crear reportes sobre clientes, préstamos y pagos.
- Facilitar el análisis de la información financiera para la toma de decisiones.

## Resumen
El sistema centraliza y automatiza las operaciones de un servicio de préstamos, mejorando la eficiencia en el registro, control y análisis de clientes, préstamos y pagos, además de proporcionar reportes claros y organizados para la administración.
